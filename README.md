# 🏠 住宅ローン金利トラッカー

住宅ローンの **変動金利** と **固定金利(フラット35)** の推移を、デスクトップで見るためのツール。

![overview](各金利をグラフで表示します)

## 使い方

- **`更新して開く.bat` をダブルクリック** → 最新金利を取得してグラフを表示。
- グラフだけ見たいとき → `index.html` をダブルクリック(前回取得したデータで開く)。

初回だけ、必要なライブラリを入れてください:

```
pip install requests beautifulsoup4 lxml
```

## 何を表示しているか

| 系列 | 中身 | 出典 |
|------|------|------|
| **変動金利(店頭)** | 短期プライムレート(最頻値)+ 1.0% で算出する店頭変動金利 | [日本銀行](https://www.boj.or.jp/statistics/dl/loan/prime/prime.htm) |
| **フラット35(最頻金利)** | 借入21年以上・融資率9割以下・団信込みの最頻金利 | [flat35.com](https://www.flat35.com/) + 参考シード |

- **変動金利**は、日銀公表の短期プライムレート変更履歴から毎回フル履歴を組み直します(自己修復)。
  2009年以降ずっと 2.475% だった実態と一致します。
  ※ 実際に借りられる金利は各銀行の優遇幅でこれより低くなります(目安: 店頭 − 約1.5〜2.0%)。
- **フラット35**は当月分しか公式公開が無いため、当月は flat35.com から自動取得し、
  過去分は `seed_flat35.json`(参考値・**編集可能**)で補います。正確な数値に直したい場合は自由に編集してください。

## 仕組み(壊れにくさの工夫)

- 2つのソースは独立して取得し、片方が失敗しても、もう片方＋前回の値で動きます。
- ネット不通・サイト構造変更時は、前回の `data/history.json` を温存します。
- グラフ下部にソース別の取得ステータスを表示します。

## ファイル構成

```
kinri-tracker/
├─ 更新して開く.bat     ← これをダブルクリック
├─ index.html           ← ビューア(オフラインで開ける)
├─ scraper.py           ← データ取得
├─ seed_flat35.json     ← フラット35の過去参考値(編集可)
├─ data/                ← 生成データ(history.json / history.js)
└─ vendor/              ← Chart.js(同梱)
```

## 定期更新したい場合

Windows の「タスク スケジューラ」で毎月 `python scraper.py` を実行するように登録すると、
バックグラウンドで金利履歴が積み上がっていきます。
